#### 기본형과 참조형
- 기본형에는 값이 들어있고, 참조형에는 위치가 들어가있다.
- 대원칙 : **자바는 항상 변수의 값을 복사해서 대입한다.** (`VariableCopy`)
- 사용하지 않는 인스턴스는 `GC`가 자동으로 메모리에서 제거한다.
